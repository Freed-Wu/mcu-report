mcu-report
=================================

A backup of my homework.

See [MusicPlayer] for the code of mcu-report.

Dependent
---------

1.  A LaTex distribution. Such as [texlive].
2.  [latex-listings-eclipse].

Install
-------

``` {.zsh}
git clone git@github.com:Freed-Wu/mcu-report
cd mcu-report
latexmk -pvc main.tex
```
