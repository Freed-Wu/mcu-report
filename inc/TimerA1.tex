Timer\_A实验
\subsection{实验目的}
定时模式-Timer\_A0，增/减计数模式，时钟源SMCLK，TA0CCR0 = 50000，ISR内翻转P4.1。
\par\indent TACLK = SMCLK = default DCOCLKDIV。
\par\indent 示波器观察P4.1输出波形，与实验5.1对比，可得出什么结论。

计算机、开发板、示波器、信号源、电源、Code Composer Studio v5、串口调试助手等。
